# AI Test Data Assistant

A practical AI solution that saves time, improves QA productivity, and scales test-data generation across projects.

---

## Problem Statement

QA and IT teams spend significant time creating realistic test data manually. Key pain points include:

- Repetitive data creation
- Inconsistent datasets
- Privacy concerns with real customer data
- Slower testing cycles

---

## Solution Overview

An AI-powered **Chrome Extension** that generates realistic synthetic customer data in seconds using **Google Gemini**.

---

## Key Features

- AI-generated customer data
- Country-specific records
- Fields: Name, Email, Phone, Address, Age
- CSV export
- Easy-to-use interface

---

## Business Impact

- Reduces manual effort
- Accelerates QA execution
- Improves testing coverage
- Eliminates dependency on real customer data

---

## Time Savings

| Method | Time Required |
|---|---|
| Manual | 10–15 mins for small datasets; hours for larger sets |
| AI Assistant | Seconds |

> **Estimated reduction in effort: 90%+**

---

## Architecture

```
Chrome Extension
      ↓
Node.js + Express Backend
      ↓
Google Gemini API
      ↓
Generated Test Data + CSV Export
```
